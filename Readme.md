# KwanNurse-Bot

KwanNurse-Bot เป็นโปรเจกต์ LINE Bot ที่ผมสร้างขึ้นเพื่อทดลองพัฒนาแชตบอทด้านสุขภาพ  
โดยมีแนวคิดหลักคือ “บอทพยาบาล” ที่ช่วยรับข้อมูลอาการเบื้องต้น  
ประเมินความเสี่ยง และช่วยจัดการเรื่องการนัดหมาย

โปรเจกต์นี้ถูกสร้างขึ้นเพื่อการเรียนรู้จริง  
ไม่ใช่แค่ให้โค้ดรันได้ แต่เพื่อฝึกการออกแบบระบบและจัดโครงสร้างโค้ดให้เป็นระเบียบ

---

## บอทนี้ทำอะไรได้บ้าง
- รับข้อมูลอาการจากผู้ใช้ผ่าน LINE
- ประเมินความเสี่ยงสุขภาพในระดับเบื้องต้น
- ช่วยจัดการและบันทึกการนัดหมาย
- แยกการทำงานเป็นส่วน ๆ เช่น routes, services, utils และ database
- เป็นโครงสร้างพื้นฐานสำหรับต่อยอดระบบสุขภาพหรือบอทในอนาคต

---

## แนวคิดของโปรเจกต์
ผมอยากทดลองสร้างบอทที่:
- ไม่ใช่แค่ตอบข้อความธรรมดา
- มี logic และ flow การทำงานที่ใกล้เคียงระบบจริง
- สามารถนำไปพัฒนาเป็นโปรเจกต์ที่ใหญ่ขึ้นได้

KwanNurse-Bot จึงเป็นเหมือนสนามทดลอง  
สำหรับฝึกการคิดเชิงระบบ การเขียนโค้ด และการจัดการโปรเจกต์

---

## โครงสร้างโปรเจกต์ (คร่าว ๆ)
- `routes/` : จัดการเส้นทางและการรับ request
- `services/` : logic หลักของระบบ
- `utils/` : ฟังก์ชันช่วยเหลือต่าง ๆ
- `database/` : จัดการข้อมูลและการเชื่อมต่อฐานข้อมูล

โครงสร้างนี้ถูกออกแบบมาเพื่อให้อ่านง่าย แก้ไขง่าย และขยายต่อได้ในอนาคต

---

## เทคโนโลยีที่ใช้
- Python
- LINE Messaging API
- Libraries ที่เกี่ยวข้อง (ดูใน `requirements.txt`)

---

## หมายเหตุเกี่ยวกับการใช้ AI
โปรเจกต์นี้มีการใช้ AI เป็นผู้ช่วยในการเขียนโค้ดบางส่วน  
แต่ผมอ่าน ทำความเข้าใจ และปรับแก้โค้ดด้วยตัวเอง  
เป้าหมายหลักคือการเรียนรู้และเข้าใจระบบ ไม่ใช่แค่ให้โค้ดทำงานได้

---

## สถานะโปรเจกต์
โปรเจกต์นี้ยังอยู่ในช่วงพัฒนา  
อาจมีการปรับปรุง แก้ไข หรือ refactor โค้ดอยู่เรื่อย ๆ  
เพื่อให้โครงสร้างและ logic ดีขึ้น

---

## ผู้พัฒนา
พัฒนาโดยผมเองคนเดียว
โปรเจกต์นี้สร้างขึ้นเพื่อการเรียนรู้และพัฒนาทักษะส่วนตัว
